# Coloc


## Description:

## Prerequisite
Docker, composer, npm.

## Composer
In `www/app` folder, run
```console
composer install
```

## Run docker run !!!

To set and start api, run inside `www` directory root :
```console
docker compose up -d --build
```

## Node
In `client` folder, run
```console
npm install
```

## Run npm run !!!
In `client` folder, run
```console
npm run start
```
